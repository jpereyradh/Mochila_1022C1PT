a. ¿Quién inventó el sistema de control de versión Git y por qué?
	Linus Torvalds  pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.
b. ¿A quién pertenece actualmente Github y por qué?
	Actualmente le pertenece a microsoft, la compra fue un paso más en la inclinación de Microsoft para enfocarse todo lo posible al desarrollo
c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
	Si se puede hacer usando GitHub Desktop
