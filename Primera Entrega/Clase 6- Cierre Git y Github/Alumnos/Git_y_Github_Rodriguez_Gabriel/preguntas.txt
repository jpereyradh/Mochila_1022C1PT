a. ¿Quién inventó el sistema de control de versión Git y por qué?

El creador de GIT fue Linus Torvalds pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

b. ¿A quién pertenece actualmente Github y por qué?

Github pertenece actualmente a Microsoft porque este busca un acercamiento a la comunidad en el desarrollo de software libre y abierto

c, ¿Hay otra forma que no sea la terminal para trabajar con Github?

Se puede instalar Github desktop que seria una interfaz grafica para traspasar información.
