¿Quién inventó el sistema de control de versión Git y porqué?
- Linus Torvalds inventó el sistema git en 2005 cuando necesitó un software para gestionar los cambios en el kernel de Linux. Desde entonces Git es el mejor software para gestionar las modificaciones del sistema de archivos de un proyecto.
